Wordpress plugin that syncs products' availability between WooCommerce and Allegro

== Description ==
This plugin may be helpful if you sale your products in both WooCommerce and Allegro. It can sync the products' quantity between your online stores when client buys a product.

== Frequently Asked Questions ==
= How to configure the plugin? =
You can get access to the configuration panel under `Tools -> Integration of Allegro and WooCommerce` in your WordPress panel.

= How to link to Allegro? =
This process is well described in the config panel. On [GitHub](https://github.com/dotandl/integration-allegro-woocommerce) you can also find some help.
